Reflection

1. What did you learn while creating your portfolio homepage?

I've acquired knowledge on how to use Bootstrap 5 for creating a responsive and visually appealing portfolio website. I've also gained a profound understanding of how containers, cards, modals, and navigation bars work together to create a cohesive layout. I also enhanced my HTML skills without relying on customized CSS while arranging my personal information in a meaningful way.

2. What challenges did you encounter while working on this activity?

Some challenges I encountered included understanding how Bootstrap components interact, particularly when customizing the navbar and modal. Ensuring that everything remained responsive across different screen sizes required significant effort, as did selecting the appropriate classes to achieve the desired spacing and alignment. In addition, I found it challenging to style the portfolio using only Bootstrap because the instructions prohibited customized CSS.

3. How will this portfolio homepage be helpful to you as a student?

This portfolio homepage acts as my personal space to highlight my work and accomplishments. As a student, it supports the development of my online presence and helps me get ready for future opportunities to share it with potential employers, mentors, or collaborators.
